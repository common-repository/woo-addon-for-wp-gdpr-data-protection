WooCommerce Addon will help you to secure all your WooCommerce user's private data with the most secure AES Encryption and Decryption with OpenSSL method.

== Description ==

**Addons for the [WP GDPR Data Protection](https://wordpress.org/plugins/wp-gdpr-data-protection/ "WP GDPR Data Protection") Plugin.**
The plugin has been designed specifically for the WP GDPR Data Protection Plugin. This will add more features in your store for WooCommerce Data.


== Installation ==
1. Install and activate the WP GDPR Data Protection plugin.
2. Unzip the downloaded file and upload the plugin folder into the wp-content/plugins/ directory of your WordPress site.
3. Activate the plugin through the ‘Plugins’ menu in WordPress.
4. Login after activation.
